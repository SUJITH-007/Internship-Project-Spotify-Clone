# Spotify Clone (MERN Stack)

A Spotify-inspired music streaming web application built using the MERN stack.  
This project is being developed as part of my internship and is currently a work in progress.

## Project Status
🚧 Work in Progress — under active development.

## Planned Features
- User authentication 
- Browse and search music
- Playlist creation and management
- Audio streaming
- Responsive UI inspired by Spotify

## Tech Stack
**Frontend:** React.js  
**Backend:** Node.js, Express.js  
**Database:** MongoDB(NO SQL)
